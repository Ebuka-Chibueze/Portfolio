Graphical Social Networking

GSN

This will be a social networking site. The user will make posts, import pictures and videos, and chat. However, unlike some social networking sites, this one will feature communication by graphics. Now the user doesn't have to be some sort of professional artist. In fact, the social canvas will be used to show demonstration during chat. The social canvas is not limited to only drawing. It will also feature simple paints tools such as the magic wand, pencil, eraser, shapes, and even textboxes. The purpose of this canvas is to show demonstrations of any form.

What does this stand alone feature has to offer from other social media websites?

- The user can draw on the social canvas which will render the same drawing on the recipient's screen as opposed to uploading pictures.
- Each drawing on the canvas is of high quality whereas pictures often times tend to be low quality and can take time to upload. It sometimes has an error when uploading pictures.
- Explanation would be made easier by easy to see demonstration on the social canvas e.g Math, Computer Science, etc.

The PHP will keep track of a user's name, username, phonenumber etc. It will keep track of the posts and chats sent through the network. The functions for PHP will keep track of posts and chat conversations made by users while the functions for the JavaScript handles all the mechanisms for the website such as redirecting to profile or editing your profile or post. There will also be a PHP or JavaScript method that will render the user's drawing on the recipient's computer.

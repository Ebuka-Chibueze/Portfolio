Welcome to Trivial Hangman

Instructions

Here's how the game works. It's basically Hangman but in this game,
instead of guessing a letter, you'll answer a trivia question. If
your answer is correct, you'll unlock a random letter of the passage.
If not, then parts of the stick man will be drawn. Like the original Hangman game,
you'll be given an option to solve the whole piece, in which case you'll
be given a prompt to type your answer. If you solved the whole passage,
then you've won the whole game. If not then the game continues with a part
of the stick man being drawn.
